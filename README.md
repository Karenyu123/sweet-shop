# 💃💃sweet-shop

#### 主要功能：

​	商品列表浏览，上拉加载更多，回到顶部

​	查看商品评论，商品参数、商品推荐等

​	多样化分类，满足不同购物选择

​	添加到购物车，可对商品进行增删改查



#### 主要技术栈：

​	vue-cli3 搭建整个项目框架

​	vuex 管理购物车数据状态

​	vue-router 实现页面跳转

​	vue-lazyload 实现图片懒加载，提高页面加载性能

​	better-scroll 控制页面滚动，上拉加载更多
